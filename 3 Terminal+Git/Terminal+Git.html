      • besides .sh (projects scripting extention) there is system files for bash (which is big aspect but in short) 2 main scriping files of differant extentions that can be customized , bashrc and bash_profile:
    • Bashrc is a configuration file for Bash shell,this file is executed each time you open a new terminal, it contains commands that are executed in every kind of shell.Bashrc is used to set environment variables, define aliases, and create functions. It's also used to prompt that appears in your terminal.
    • Bash_profile is also configuration file for Bash shell, but it's executed only once when you log in to your account. its like few more commands that special for each user and they dont take effect in a sission untill a user log-in . This file is used to set the user environment, variables and run commands that are needed only once at start of your session/subshell. Bash_profile is commonly used to set PATH variable, which is variable that determines where shell searches for custom executables.
      • Bashrc is executed each time you open a new terminal window regardless if its on account log-in or not , while bash_profile is executed only once when you log in to your account. This means that any changes you make to bashrc will take effect immediately, while when you log out and log back in again.
      changes to bash_profile will take effect only Bashrc is commonly used to set aliases, define functions, and customize prompt. Bash_profile s commonly used to set PATH variable and to run commands that are needed only once at start of your session.
      Suppose you want to set an alias for ls command to list files in long format. You can do this by adding following line to your bashrc file: alias ll='ls -l' this will be available for all terminal windows .
      Suppose you have installed a custom application in your home directory, and you want to add its location to your PATH variable so that you can run it from anywhere in your shell. You can do this by adding following line to your bash_profile file: export PATH=$PATH:/home/user/myapp/bin .
