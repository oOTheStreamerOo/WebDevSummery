  • Source Code: is a set of human-readable instructions written in a programming language. It can include comments and produces an output when run. Its written in high-level languages like C, C++, Java, Python, JavaScript, etc. Once written, its saved in a file and then converted into machine-understandable code, known as object code, by a language translator.
    Object Code (Compiled Code): This is low-level, machine-understandable code that is the output of a compiled source code. Its not easily understandable by programmers unlike source code. Its stored in object files like COM files, COFF, etc. Object code, machine code, and binary code are often used interchangeably.
